**Open Modbus** is an open-source initiative focused on creating reliable, fully transparent hardware and firmware for Modbus-based automation systems.  
Our projects include digital/analog I/O modules, STM32-based controllers, and tools for building distributed RS-485 networks.

## What We Provide
- Open hardware (schematics, PCB, BOM)
- Open firmware (Modbus RTU, STM32)
- Documentation and wiring guides
- Ready-to-use industrial I/O modules (e.g., **OM-64DO**)
